GitHub menyediakan layanan hosting web gratis bernama **GitHub Pages**. Anda bisa mempublikasikan kode HTML hasil generator di atas menjadi website aktif yang bisa diakses langsung lewat handphone atau browser Anda.

1

Buat Akun dan Repositori GitHub Baru

Jika belum memiliki akun, daftar di github.com. Klik tombol **New** di dashboard Anda untuk membuat repositori baru. Beri nama repositori Anda sesuka hati, atur ke **Public**, lalu klik **Create repository**.

2

Unggah File index.html

Di dalam tab **Generated Code** pada aplikasi ini, klik tombol **Salin Kode HTML**. Di halaman repositori GitHub baru Anda, klik tautan **creating a new file**, beri nama file tersebut dengan nama index.html, lalu *paste* seluruh kode yang disalin ke dalamnya. Scroll ke bawah dan klik **Commit changes**.

3

Aktifkan Fitur GitHub Pages

Di repositori GitHub Anda, pergi ke menu **Settings** (ikon roda gigi di tab atas) -> **Pages** (pada sidebar sebelah kiri). Di bagian *Build and deployment*, ubah *Branch* dari **None** menjadi **main** (atau master), biarkan folder tetap / (root), lalu klik **Save**.

4

Website Anda Selesai!

Tunggu sekitar 1 hingga 2 menit. Refresh halaman Settings Pages Anda, GitHub akan menampilkan tautan hijau di bagian atas yang berbunyi: **"Your site is live at..."**. Klik tautan tersebut dan website trading pipeline kustom Anda sudah aktif mengudara secara publik!
